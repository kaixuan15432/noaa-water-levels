# NOAA Water Levels Dashboard

A web-based dashboard for visualizing and comparing NOAA water level observations with model predictions for Florida coastal stations.

## Features
- Interactive map with 12 Florida tide stations
- Data comparison: NOAA observed vs model predictions  
- Anomaly removal (mean subtraction)
- Mean Absolute Error (MAE) calculation
